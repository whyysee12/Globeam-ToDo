# Quick Start Guide - New Features

## 1. Adding a Remark (Work Type) 🏷️

**Where:** Tasks page → Add a task form

When creating a task, you'll see a new field: **"Remark (Type of work)"**

Examples:
- Documentation
- Team Meeting
- Code Development
- Client Call
- Research

This helps you and your team understand what kind of work each task is.

---

## 2. Editing Tasks ✏️

**Where:** Any task card → Blue pencil icon

Click the **Edit** button to modify any aspect of a task:
- Title
- Description
- Remark
- Task Type
- Priority
- Due Date

Then click **Update** to save your changes.

---

## 3. Filtering & Sorting Tasks 📊

**Where:** Dashboard page → Filter and Sort cards

### Sort By:
- Latest (newest tasks first)
- Name (alphabetical A-Z)
- Priority (High → Medium → Low)
- Priority (Low → Medium → High)

### Filter By:
- All Tasks
- High Priority Only
- Medium Priority Only
- Low Priority Only
- Today's Tasks Only
- Regular Tasks Only
- Fixed Tasks Only

**Tip:** Use combinations of filters and sorts for powerful task organization!

---

## 4. Task Type Selector 📋

**Where:** Tasks page → Add/Edit form

Choose which type of task you're creating:

- **Today's Task** - One-time work for today
- **Regular Task** - Recurring work you do often
- **Fixed Task** - System-assigned daily tasks

This helps categorize your workflow!

---

## 5. Due Dates (Not Times!) 📅

**Where:** Tasks page → Due Date field

Instead of just a time, now set a full **date** for when tasks are due:

- Better for planning across days
- Supports multi-day tasks
- Easier to manage deadlines

---

## 6. View Previous Day Tasks 🕐

**Where:** Dashboard page → Date navigation

See tasks from any previous day:

1. Use **← arrow** to go to previous day
2. Use **date picker** to jump to specific date
3. Use **Today** button to return to today
4. Use **→ arrow** to move forward (if before today)

View all your task history!

---

## 7. Export Tasks to Excel 📥

**Where:** Dashboard page → Export buttons

### Two Export Options:

**Export Day:**
- Exports current date's tasks only
- Perfect for daily reports

**Export All:**
- Exports all filtered/sorted tasks
- Respects your current filters and sort order

The file downloads as CSV (opens in Excel, Google Sheets, etc.)

---

## Task Summary Examples

### Daily Summary Report
1. Go to Dashboard
2. Select a date
3. Click "Export Day"
4. Open CSV in Excel
5. Print or email report

### High Priority Tasks Report
1. Go to Dashboard
2. Filter by "High Priority"
3. Click "Export All"
4. Share with team

### Previous Week Review
1. Select previous day's date
2. Apply filters as needed
3. Export for documentation

## Keyboard Shortcuts

- **Tab** - Move between form fields
- **Enter** - Submit form (Add/Update)
- **Escape** - Cancel edit form

---

## Troubleshooting

**Q: Can I edit a fixed task?**
A: Yes! You can edit any task, including its remark, priority, and due date.

**Q: What format is the export?**
A: CSV format (opens in Excel, Google Sheets, and other spreadsheet apps)

**Q: Can I see tasks from a year ago?**
A: Yes! Use the date picker to navigate to any past date.

**Q: Do remarks show on the home page?**
A: Yes! Remarks appear on task cards with a 💡 emoji icon.
